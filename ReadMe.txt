Weather App is a Web application that display's user the current weather of the city name entered. User can enter any city name and can see it's weather.
User can also see next seven days weather forecast of the selected city and plan their activities.

How To Set it Locally:
Download ZIP file after clicking on Code option.
Extract files on your PC.
Open Weather App project in VS code.
Open terminal and run commad "npm start".
Your react app will be running.
